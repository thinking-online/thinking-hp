# 成蹊大学 経営学部 英語 完全攻略マニュアル — 第3部 大問別攻略ドリル（ビルド）

学部別合格設計塾THINKING（合同会社ARC）／大学別・学部別攻略マニュアルシリーズ。
成蹊大学 経営学部の一般選抜（A方式）英語・全5大問を **大問ごとに1冊のPDF** として攻略する
「第3部 大問タイプ別攻略ドリル」の生成環境。実際の過去問は転載せず、**すべて完全オリジナル**の
成蹊型演習問題＋詳解で構成する。

## 成果物（`dist/`）

| 巻 | 大問 | 形式 | 収録 |
|----|------|------|------|
| 第3部-Ⅰ | 大問Ⅰ 会話文Set選択補充 | 会話2本・語群Set A/B | オリジナル12セット（会話24本・空所156）＋全空所詳解＋会話定型表現100 |
| 第3部-Ⅱ | 大問Ⅱ 総合長文読解 | 下線言い換え/真偽/整序/語法/一致 | オリジナル長文8本＋全設問詳解＋言い換え同義語テーブル |
| 第3部-Ⅲ | 大問Ⅲ 英問英答長文 | 設問文がすべて英語 | オリジナル長文11本＋全設問詳解＋英語設問定型フレーズ50 |
| 第3部-Ⅳ | 大問Ⅳ 長文空所補充 | 短文2本・語彙空所＋文選択空所 | オリジナル17セット＋全空所詳解＋論理マーカー一覧 |
| 第3部-Ⅴ | 大問Ⅴ 短文空所補充 | 語法・イディオム・準動詞・時制 | 分野別オリジナル短文問題＋全問詳解＋紛らわしいペア一覧 |

各巻とも A4・WeasyPrint製・60ページ級。THINKING標準デザイン（navy/gold/落款）。

## ビルド方法

```bash
cd build/seikei
python3 build_daimon1.py   # 大問Ⅰ
python3 build_daimon2.py   # 大問Ⅱ
python3 build_daimon3.py   # 大問Ⅲ
python3 build_daimon4.py   # 大問Ⅳ
python3 build_daimon5.py   # 大問Ⅴ
```

出力は `dist/成蹊大学_経営学部_英語_完全攻略マニュアル_第3部-N_….pdf`。

### 依存
- Python3 ＋ WeasyPrint（`pip install weasyprint --break-system-packages`）
- フォント：Noto Sans / Serif CJK JP（`fonts-noto-cjk` / `fonts-noto-cjk-extra`）
- QA用：poppler-utils（`pdfinfo` / `pdftoppm`）

## 構成

- `seikei_common.py` … 全巻共通のデザインシステムCSS・表紙・章扉・枠・表・レンダラー
- `reading_common.py` … 長文読解系（Ⅱ・Ⅲ・Ⅳ）共通の本文/設問/解説レンダラー。
  本文タグ `[[u:N]]…[[/u]]`＝下線部（設問N）、`[[b:N]]`＝空所。設問は正解を先頭に置き
  `answer="ア"` とすれば、`distribute()` が設問番号から決まる位置へ正解を分散配置する。
- `build_daimonN.py` … 各巻のビルド本体（前付け・戦略章・問題編・解説編・付録の組み立て）
- `dN_sets/` … 各巻の演習データ（1ファイル＝1セット/1長文/1カテゴリ）
- `dN_appendix.py` … 各巻の付録
- `daimon1_*.py` … 大問Ⅰのデータ（`daimon1_data` 前付け・戦略、`d1_sets/` 演習、`daimon1_appendix`）

## 設計原則

- 実過去問の英文・設問・選択肢は一切転載しない（分析はメタ情報のみ、演習は完全オリジナル）
- 正解記号は `distribute()` で ア/イ/ウ/エ に自動分散（先頭固定の偏りを排除）
- 分析事実は2025年度実物で確認した構造に基づく。配点・試験時間は赤本で最終照合する前提
