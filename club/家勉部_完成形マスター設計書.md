# 家勉部ポータル 完成形マスター設計書 v1

このドキュメントは家勉部ポータルが最終的にどうなるかの全体像。
個別機能の実装は、これを正典として各機能の設計書・指示書に展開する。

---

## 0. コンセプト

「一人では続かない勉強が、続いてしまう仕組み」を、ポータル上で完成させる。
中心にいるのは **家勉くん**(マスコット犬のAIコーチ)。
生徒は家勉くんと一緒に計画を立て、毎日タスクをこなし、犬を育て、リーグで競う。
仮入部の7日間でこのループを体験し、「これを失いたくない」状態をつくって入部に繋げる。

---

## 1. 完成形の全体像(生徒が見る画面)

ポータルを開くと、上から:

1. **仮入部カウントダウン**(仮入部の人だけ): 「仮入部 3日め / 残り4日」+正式入部ボタン
2. **家勉くん**: AIコーチへの入口。話しかけると計画・励まし・勉強法を返す
3. **マイステータス**(既存): 犬のランク・累計日数・プログレスバー・ストリーク
4. **今日のタスク**(新規): 今日やる分。チェックで達成率が動く。全部終わると「やりきった!」演出
5. **やることリスト**(新規): 1週間/テスト期間の宿題。家勉くんが日割りに整理
6. **勉強時間の記録**(既存): タスク完了と連動
7. **予選リーグ / 全国大会**(既存 v5): 競争
8. **小テスト範囲・小テストランキング・限定コンテンツ**(既存)

情報量が増えるので、**タブ構成**を導入する(下記7章)。

## 2. 家勉くん(AIコーチ)— 中心機能

### 役割
ポータルに常駐するAIコーチ。マスコット犬の人格で喋る。

### できること
- 計画相談: 「テストまで5日、何やる?」→ やることリストと残り日数を見て計画提案
- 励まし: 「サボった」→ 詰めずに寄り添い、明日へ繋ぐ
- 勉強法: 想起頻度メソッド等(家勉部の知識)を踏まえて回答
- タスク整理: 漠然とした不安を具体的タスクに変換

### 家勉くんが「自分を分かってくれてる」理由
GASが、その生徒のデータ(累計日数・今週の勉強時間・ストリーク・リーグ順位・やることリスト・今日のタスク達成率)を取得し、Claudeへの入力に含める。
→ 「今週もう3日連続だね」「リーグあと30分で2位だよ」のように文脈を踏まえて喋れる。
これがただのAIチャットとの決定的な差。

### 家勉くんの人格(後でチューニング)
てったの教育観・口調・励まし方を system プロンプトに乗せる。
詰めない/背中は押す/関西弁まじりのフランクさ/締めは「じゃ、やってこ。」
この人格定義は GAS 内に置き、運用しながら調整する。

### 技術構成(キーの安全性)
```
生徒のポータル
  → 家勉部GAS(APIキーを保管 + 生徒データ取得 + 人格プロンプト付与 + 利用回数制限)
    → Claude API
  ← 家勉くんの返事だけが生徒に届く
```
- **APIキーは絶対にHTML/ポータルに書かない**。GASのスクリプトプロパティに保管。
- 生徒からはキーが一切見えない。既存のLIFT認証で「会員だけ」が使える。
- モデルは Claude Haiku 4.5($1/$5 per Mtok)を基本にする。計画整理・励ましには十分。
  必要なら Sonnet 4.6($3/$15)へ切替可能に(設定で持つ)。

## 3. やることリスト + デイリータスク — 毎日やりきる仕組み

### レイヤー1: やることリスト(中期の宿題)
- 1週間 or テスト期間でやることを登録(例「英単語1-500」「数学ワークp30-60」)。
- 追加・削除できる。期間中は残る。

### レイヤー2: 家勉くんが日割りに整理
- やることリスト + テストまでの日数 + 1日に使える時間を家勉くんに渡すと、
  「何日に何をやるか」に振り分けてくれる。
- 「何から手をつければいいか分からない」を解消。これが価値の核。

### レイヤー3: 今日のタスク(やりきる)
- その日やる分が「今日のタスク」として並ぶ。自分で追加も可。
- チェックすると達成率が動く(3個中1個→33%)。
- **全部チェックで「今日のミッション完了!」演出**(犬が喜ぶ)。
- 量は「確実にやりきれる現実的な量」に。やりきれない量で心を折らない。
- できなかったタスクは翌日に優しく繰り越し。責めない。

### レイヤー4: 既存システムと接続
- タスクのチェック時に「何分やった?」を軽く添える → 勉強時間として記録。
- その時間が犬(ランク)・リーグ・全国に反映。タスク完了と記録が1つの流れに。

## 4. 仮入部カウントダウン

- 会員マスタの「仮入部期限」(承認日+7日)から残り日数を計算。
- 仮入部の人にだけ表示。
- 残り日数で煽りを変える(残り1日は「今日が仮入部最終日!」と強調 + 正式入部フォームへ)。
- 正式入部フォームURLは06_設定で管理。

---

## 5. データ設計(何百人でも回る)

スプレッドシートDBに追加:
- **11_やることリスト**: member(line_user_id), id, 内容, 期間ラベル(今週/テスト名), 状態(未/完了), 追加日
- **12_デイリータスク**: member, 日付, id, 内容, 完了フラグ, 分(任意), 由来(自分/家勉くん)
- **13_家勉くん会話ログ**(任意): member, 日時, 役割(user/assistant), 本文 — 文脈保持と乱用監視用。保存日数を限定。

1人1日数タスク+数会話なら、40人で1日数百行。何百人でもスプレッドシートで耐える。
利用回数制限(家勉くん1日◯通、計画整理1日◯回)を06_設定で持ち、GASでカウント。

## 6. 費用(2026年6月時点の実料金で試算)

- Haiku 4.5: $1/$5 per Mtok。1往復 ≒ 1〜2円。
- Sonnet 4.6: $3/$15 per Mtok。1往復 ≒ 4円前後。
- 40人 × 1日数回 × Haiku ≒ 月数千円。サブスク(月1万円超)で十分ペイ。
- 暴走対策: 1人1日の上限回数をGASで強制。バグ・乱用での膨張を防ぐ。

機能が増えるため、縦長1ページからタブ構成へ。
- **ホーム**: カウントダウン / 家勉くん入口 / マイステータス(犬) / 今日のタスク
- **計画**: やることリスト / 家勉くんの計画整理
- **きろく**: 勉強時間の記録 / 小テスト範囲
- **大会**: 予選リーグ / 全国大会 / 小テストランキング
- **部室**: 限定コンテンツ / 自習室

(タブ名・配置は実装時に微調整。既存セクションは消さず移動するだけ。)

## 8. 段階リリース計画

### 第1段階: 家勉くん(AIコーチ)
- GASにClaude API接続(キーをスクリプトプロパティに保管)。
- 家勉くんは生徒データ(犬・リーグ)を見て喋れる。
- ポータルにチャットUI。
- まず「家勉くんと会話できる」を完成させる。最も感動が大きい。

### 第2段階: やることリスト + デイリータスク
- 11/12シート追加。登録・チェック・達成率・やりきった演出。
- 家勉くんが計画をタスクに落とす連携。
- 勉強時間記録との連動。

### 第3段階: 仮入部カウントダウン + タブ化 + 仕上げ
- カウントダウン、正式入部導線。
- 画面をタブ構成へ再編。

### 既に完成
- 犬ランク(v4)、予選リーグ・全国大会(v5)。適用すれば動く。
