# 設計方針書(設計担当の成果物)

対象:日本人中高生/教材:英文60文/目的:「訳の丸暗記」と「構造・意味の真の理解」の判別

## 1. テストで確認すべき観点リスト

各観点に【証拠性】=なぜそれが理解の証拠になるかを添える。

### A. 文構造の把握

- **A1. 主語(S)と述語動詞(V)の特定**(特に主語が長い文・命令文・従属節内)
  【証拠性】訳文の暗記では「どの語がどの語に対して述べられているか」を問われた瞬間に答えられないため、SV対応を指させることは構造把握の直接証拠になる。
- **A2. 節の切れ目と役割の特定**(この when 節はどこからどこまでか/副詞節か名詞節か)
  【証拠性】丸暗記した訳は文全体を一塊で覚えているだけなので、部分の境界と機能を答えられるのは自力で文を分解した証拠になる。
- **A3. 修飾関係の特定**(この副詞・前置詞句・分詞はどの語を修飾するか)
  【証拠性】訳文には修飾の矢印が明示されないため、係り先を正しく指せることは英文側を読んだ証拠になる。
- **A4. 句の内部構造**(to 不定詞は目的か結果か、if は「もし」か「〜かどうか」か)
  【証拠性】同じ形の語(if, that, as など)の多義を文脈で選び分けられることは、形→意味の変換を自分で行っている証拠になる。

### B. 指示・照応・省略

- **B5. 指示語・代名詞の指す内容**(it, the other person, one などが具体的に何を指すか)
  【証拠性】訳文の中では指示語が自然に訳出されて消えていることが多く、原文側で指示対象を特定できるのは英文を追跡している証拠になる。
- **B6. 省略・共通要素の補完**(命令文の隠れた主語 you など)
  【証拠性】書かれていない要素を補えるのは、文法規則から文の完全な形を復元できている証拠になる。

### C. 語彙・表現の文脈内理解

- **C7. 多義語の文脈内語義**(pass through, reach, keep O on 〜 がこの文では何を意味するか)
  【証拠性】単語帳の第1義ではなく文脈に合う語義を選べることは、文全体の意味から逆算して読めている証拠になる。
- **C8. 機能語・文法項目の働きの説明**(until と by the time の違い、some/any の使い分けなど)
  【証拠性】「なぜこの形なのか」を説明できることは、訳の結果ではなく仕組みを理解している証拠になる。

### D. 意味の再構成(訳への依存を断つ)

- **D9. 訳の根拠説明**(「なぜ『〜かどうか』と訳すのか」を1文で言わせる)
  【証拠性】暗記した訳は再生できても正当化できないため、根拠の言語化は理解と暗記を最も直接に分離する。
- **D10. パラフレーズ・場面説明**(この文の指示内容を、訳文と違う自分の言葉で説明させる)
  【証拠性】表層の訳文と異なる表現で同じ内容を言えるのは、意味表象そのものを持っている証拠になる。
- **D11. 部分訳・スロット訳**(文全体でなく、下線部だけ・節1つだけを訳させる)
  【証拠性】全文訳を丸暗記していても部分の切り出しには対応できないため、部分単位の訳は自力の解釈を要求する。
- **D12. 変形への追従(転移)**(同じ構造で語彙だけ替えた新出文を読ませる)
  【証拠性】未見の文に構造知識を適用できることは、その文固有の暗記ではなく一般化された理解の証拠になる。

### E. 真偽・推論

- **E13. 内容真偽・条件関係の理解**(訳文の字面から即答できない推論問題)
  【証拠性】条件・時間関係(if / until / when)の論理を運用できるのは、文の命題内容を処理している証拠になる。

> 運用上の優先度:全観点を毎文問うのではなく、**その文の「解釈上の山場」に対応する観点を2〜3個選ぶ**。

## 2. 避けるべきアンチパターン(丸暗記で正解できてしまう問題)

1. **全文和訳をそのまま書かせる** — 配布済みの訳を再生するだけで満点になる。
2. **訳文の穴埋め** — 英文を見ずに、覚えた日本語の復元だけで解ける。
3. **選択肢が訳文の字面照合で切れる4択** — 正解肢は必ず配布訳と別の表現にする。
4. **単語の意味を単独で問う** — 単語帳知識で解け、文構造の理解を経由しない。
5. **暗記済みの訳から答えが透ける設問** — 設問を作ったら必ず「配布訳だけを見て解けるか?」を自問する。

**判定基準(全設問共通)**:「英文を隠して配布訳だけ渡された生徒が正解できるか?」— Yes ならその問題は不合格。

## 3. 分量と問題形式(60文で運用可能な構成)

**1文あたり3問・約3〜5分**を標準とする。60文なら10文×6回(1回30問・約40分)に分割可能。

| 問 | 形式 | 内容 | 採点 |
|---|---|---|---|
| 問1 | 指差し/記号 | 構造の特定(SV・節の範囲・係り先・指示語) | 客観・即採点 |
| 問2 | 短答選択(3〜4択)または一語答 | 機能・語義の判別(節の役割、多義語、省略の補完、真偽判定) | 客観・即採点 |
| 問3 | 短記述(1〜2文) | 根拠説明・部分訳・パラフレーズ | ルーブリック採点 |

- 3問は必ず**異なる観点**(A〜C群から1つ+D/E群から1つを目安)から出す。
- 記述は1文につき1問まで(採点負荷の上限)。難所の多い文は4問まで、単純な文は2問まで可。
- 転移問題(D12)は毎文ではなく、テスト1回につき2〜3題を巻末の総合問題として置く。

## 4. 作成担当への作成仕様書(チェックリスト)

### 事前分析(問題を書く前に)
- [ ] 対象文の「解釈上の山場」を2〜3個特定したか(長い主語/多義語/指示語/省略/係り受けの紛れ/条件・時間の論理)
- [ ] 各山場を観点リスト(A1〜E13)のどれで問うか決めたか
- [ ] 山場のない観点を無理に問うていないか

### 出題構成
- [ ] 1文につき2〜4問(標準3問)/3問が異なる観点/記述は1問以内

### 丸暗記対策(全設問必須)
- [ ] **「配布訳だけを見て解けるか」テストを実施し、解ける設問を排除したか**
- [ ] 全文和訳の再生・訳文穴埋めがないか
- [ ] 正解肢が配布訳と同じ言い回しになっていないか
- [ ] 単語の意味を文脈から切り離して問うていないか
- [ ] 記述の模範解答が配布訳のコピーにならないか

### 設問の書き方
- [ ] 問う対象を英文側に下線・番号で明示したか
- [ ] 誤答肢が「ありがちな誤読」を反映しているか(でたらめな誤答は禁止)
- [ ] 記述問題に字数目安と字数の数え方を付けたか
- [ ] 設問文は中高生が一読で意図を取れる日本語か
- [ ] **設問間干渉がないか(ある問の設問文・選択肢が別の問の答えを配っていないか)**

### 解答・採点資料
- [ ] 全問に模範解答/記述に採点基準(必須要素+配点計算法+グレー答案の線引き)/各問に誤答分析メモ
- [ ] **模範解答自身が字数・文数の制約を満たしているか**

### 分量・運用
- [ ] 1文3〜5分以内/1回のテストが40分以内・記述10問以内
- [ ] 客観問題は生徒の自己採点が可能な形式か
