# 試作 第1問(最終版)

対象文:

> **When you pass through the door, <u>(1) look</u> quickly behind you <u>(2) to see if anyone is following</u>. If there is someone following, <u>(3) keep</u> your hand on the door until the other person reaches it.**

配布訳:「ドアを通り抜けるときは、誰かついて来ていないか、素早く後ろを確認しよう。もし誰かついて来ていたら、その人がドアに届くまで手を添えておこう。」

## 事前分析:この文の解釈上の山場

| # | 山場 | 出題 |
|---|---|---|
| A | **anyone / someone の使い分け**(1文目 anyone、2文目 someone。配布訳ではどちらも「誰か」で潰れており、訳照合が原理的に不可能) | 問1 |
| B | **命令文の主語**(look / keep は you への命令。配布訳「〜しよう」は we と誤解させやすく、訳依存の生徒ほど間違える) | 問2 |
| C | **see if 〜 の if(「〜かどうか」)と否定形の意訳**(「ついて来ていないか」の否定は英文に存在しない) | 問3 |

- 指示語 **it**(= the door)は、配布訳に「ドアに届くまで」と訳出済みで訳照合だけで解けるため**不採用**。
- 「to see の係り先(look)」は、直前動詞に付けるだけの位置のカンで正解でき判別力がないため、初稿から**差し替え**た(検証ログ参照)。

## 問題用紙パート

**次の英文について、下の問いに答えなさい。**(目安時間:4〜5分)

> When you pass through the door, <u>(1) look</u> quickly behind you <u>(2) to see if anyone is following</u>. If there is someone following, <u>(3) keep</u> your hand on the door until the other person reaches it.

**問1** 1文目では *if **anyone** is following*、2文目では *If there is **someone** following* と、日本語ではどちらも「誰か」にあたる語が使い分けられている。この使い分けの理由として最も適切なものを記号で選びなさい。

- ア 1文目は、ついて来る人がいるかまだ分からない場面で使われ、2文目は、実際にいる場合を想定した場面で使われているから。
- イ anyone は大勢の人を表し、someone はただ1人を表すから。
- ウ someone より anyone のほうがていねいで固い言い方だから。
- エ 同じ単語の繰り返しを避けるために言いかえただけで、意味の違いはないから。

**問2** 下線部(1) *look* と(3) *keep* の動作を行うのは誰か。最も適切なものを記号で選びなさい。

- ア この文の読み手(聞き手)であるあなた(you)
- イ 話し手と読み手を含む私たち(we)
- ウ 文中の anyone(ついて来ているかもしれない誰か)
- エ 文中の the other person(ついて来ているその人)

**問3** 配布訳では下線部(2)が「誰かついて来て**いないか**」と否定の形で訳されているが、英文中に not などの否定語は**ない**。

- (a) 下線部(2)を、英文に合わせて**肯定の形のまま**直訳しなさい。(15〜25字。句読点も1字と数える)
- (b) 英文に否定語がないのに配布訳のような訳が成り立つ理由を、**1文で**説明しなさい。(40〜60字。句読点も1字と数える。if などの英単語は1語=1字と数える)

---

## 解答・解説パート

### 問1

- **模範解答:ア**
- **解説:** any は「いるかいないか分からない・ゼロかもしれない」不確定の場面(疑問・否定・確認)で使い、some は「いる」ことを前提・想定した場面で使う。1文目は「いるかどうかを確かめる」文脈なので anyone、2文目は「(確認の結果)実際にいる場合」を想定した条件なので someone。
- **誤答分析メモ:**
  - **イ(数の違い)** … some/any の使い分けを単数・複数の問題と誤解している。-one が付けば両方単数であることも見えていない。
  - **ウ(ていねいさ)** … 語感の当てずっぽうで答えており、some/any の文法的な使い分け規則を知らないサイン。
  - **エ(言い換えただけ)** … 「英語は繰り返しを嫌う」という知識の誤適用。単語の交替には意味上の理由があるという意識がなく、英文を「なんとなく」読んでいるサイン。

### 問2

- **模範解答:ア**
- **解説:** look も keep も動詞の原形で文が始まる**命令文**であり、動作を行うのは相手=you。配布訳の「〜しよう」は命令口調を和らげた訳し方にすぎず、Let's(=we)の意味ではない。
- **誤答分析メモ:**
  - **イ we** … 訳文の「〜しよう」の字面に引きずられた典型的な訳依存の誤答。英文が原形始まりの命令文であることを確認していない。**この問題の判別核**(訳だけの生徒はイに誘導される)。
  - **ウ anyone** … 従属節内の主語と主節の(省略された)主語を区別できていない。下線部に近い名詞に飛びついている。
  - **エ the other person** … keep の動作主(手を添える側)と reaches の動作主(ドアに達する側)の対立が読めていない。

### 問3

- **模範解答:**
  - (a) 「誰かがついて来ているかどうか確かめるために」(21字)
  - (b) 「この if は『もし』ではなく『〜かどうか』の意味で、『〜ていないか』は肯定の英文を自然な日本語に言い換えた意訳だから。」(1文・59字)
- **採点基準:** ①②③各1点、計3点。満たした項目数をそのまま得点とする。ただし**①は必須要素:①が不可の場合、問3全体を0点とする**(①なしの部分点は、訳の書き写し+日本語力だけで作れてしまうため)。
  1. **(a)で、肯定の形+疑問の形の直訳ができている** —「(誰かが)ついて来ている/後をつけて来ている」等の肯定形に「〜か/〜かどうか」が付いていれば可。表現の言い換え(「尾行しているかどうか」等)は可。**否定形(「〜ていないか」等)で訳したものは設問の指示に反するため①不可**。「if=もし」と訳したものも①不可。
  2. **(a)で、下線部(2)の内容を「確かめる/見る/確認する」+「ために」と訳せている** — 対象は「ついて来ているか」であること。**単なる「後ろを見るために」は不可**(訳の「確認」を写しただけの可能性)。「会う」は誤訳で不可。「わかるために」は目的の「ために」が正しく訳せていれば許容。
  3. **(b)の説明に、次の少なくとも一方の趣旨がある** — (i)「この if は『もし』ではなく『〜かどうか』の意味」/(ii)「否定は英文にはなく、日本語側の自然な言い換え(意訳)」。**「if=もし」を前提とした説明は③不可**。字数は指定±3字まで許容。
- **誤答分析メモ:**
  - (a)を「もし誰かがついて来ているなら…」… 名詞節の if と条件の if(第2文の If)を区別できていない。**最頻出の誤読**。
  - (b)で「not が省略されている」… 英文と訳語が常に一対一対応するという思い込み。see if 〜=「〜か(どうか)確認する」の理解が欠けている。
  - (a)に配布訳をそのまま書き写す … 訳の再生で済ませようとしており、①不可 → 0点として判別できる。

### 最終セルフチェック(配布訳だけで解けるかテスト)

- **問1:** 配布訳では anyone / someone がどちらも「誰か」で、訳からは使い分けの存在自体が見えない。→ **合格**
- **問2:** 配布訳の「〜しよう」はむしろ誤答イ(we)へ誘導する。英文の原形始まりを見ない限り正解できない。→ **合格**
- **問3:** 「英文に否定語がない」という事実と肯定形の直訳は英文を参照しない限り書けない。①必須化により、訳の書き写し+日本語力での部分点も遮断。→ **合格**
