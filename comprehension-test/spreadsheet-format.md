# スプレッドシート形式仕様(Googleフォーム量産用)

全問客観式(4択)。1行=1設問。この形式でスプレッドシートに貼り付け、`gas/form-generator.gs` を実行するとクイズ形式のGoogleフォーム(自動採点・解答/解説紐づけ)が一括生成される。

## シート名: `questions`

| 列 | 列名 | 内容 |
|---|---|---|
| A | `sentence_id` | 英文の通し番号(1〜60)。同じ番号の行が1つのセクションにまとまる |
| B | `sentence_en` | 対象英文(下線などの装飾は不可。プレーンテキスト) |
| C | `sentence_ja` | 配布訳 |
| D | `q_no` | その英文内での設問番号(1〜) |
| E | `tag` | 観点タグ(例: 語法(some/any) / 構造(命令文の主語) / 用法識別(if) / パラフレーズ(until)) |
| F | `question` | 設問文 |
| G | `choice_a` | 選択肢A |
| H | `choice_b` | 選択肢B |
| I | `choice_c` | 選択肢C |
| J | `choice_d` | 選択肢D |
| K | `correct` | 正解(A/B/C/D のいずれか1文字) |
| L | `points` | 配点(整数) |
| M | `explanation` | 解説。フォームの解答フィードバック(正解時・不正解時共通)に表示される。生徒向けの言葉で2〜3文 |
| N | `note_a` | 指導者用メモ: Aを選ぶ生徒の誤解(正解肢の場合は「正解」と記載) |
| O | `note_b` | 同上(B) |
| P | `note_c` | 同上(C) |
| Q | `note_d` | 同上(D) |

- 1行目はヘッダー行(上記の列名)。
- `note_a`〜`note_d` はフォームには出力されない(誤答分析・指導用)。
- 英文中で設問が参照する箇所は、`question` 側で引用する(例: 「1文目の *to see if anyone is following* の if と同じ…」)。フォームでは下線が使えないため。

## 品質基準(全行共通)

1. 正解が1つに定まること(誰が採点しても同じ)。
2. 「英文を隠して配布訳だけで解けるか?」→ Yes なら不合格。
3. 誤答肢は「ありがちな誤読」を反映(でたらめ禁止)。選択肢の長さ・体裁を揃える。
4. 設問間で答えのリークがないこと。

## 運用フロー

1. 作成チーム(作成→一意性チェック+妥当性チェック→統合→再検収)を通過した問題だけをシートに追加。
2. `gas/form-generator.gs` をスプレッドシートの Apps Script に貼り付けて `createFormsFromSheet` を実行。
3. 生成されたフォームURL(編集用・回答用)が `forms` シートに書き戻される。
