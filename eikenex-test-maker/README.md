# 英検EX準1級 空所補充テスト データ

全1900項目（単語1600＋句動詞・熟語300）に対する4択・空所補充問題の完成データ。

## ファイル
- `eikenex_cloze_all.csv` … 本体。全1900問。ヘッダー付き。
- `eikenex_words.tsv` … 見出しマスター（No+見出し[+意味]、内部参照用）。

## 列
`No.,単語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`

## 品質方針
正確さ／唯一解（ダミーは入れると必ず破綻）／自然さ の3段検証を全項目に適用。
単語=1語空所＋1語4択／句動詞・熟語=句空所＋紛らわしい別熟語4択。
例文・和訳・意味はすべて新規書き起こし。答え位置は1〜4に均等化。
