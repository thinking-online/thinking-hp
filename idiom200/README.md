# この夏の必修 英熟語200

熟語帳5冊（ターゲット英熟語1000／速読英熟語／英検準1級／英検2級／英検準2級）を横断し、
表記ゆれを正規化した上で「複数の熟語帳に共通して載っている＝重要」という基準で頻度上位200を抽出。

- `idiom200_cloze_all.csv` … 200問の4択空所補充テスト（既存の3段検証済み問題を流用）
