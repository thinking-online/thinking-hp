# kakomon/ — 大学別過去問・問題集PDF

各大学フォルダに英語問題の過去問・演習・模試PDFを置く置き場です。
`pdfs/` と同様、Git push 後に Netlify でそのまま公開されます。

## 公開URL

```
https://<本ドメイン>/kakomon/<大学フォルダ>/<ファイル名>.pdf
```

例: `kakomon/fukuoka-u/exam-2026-early.pdf`

## ファイル名ルール

**大学名はフォルダで分かれるので、ファイル名には入れない。**
大問別の資料は **`q1` `q2` … を先頭** に置いて、一覧でパッと判別できるようにする。

### 大問別演習・攻略

```
q1-<内容>.pdf
q2-<内容>.pdf
```

例:
- `q1-reading.pdf` — 大問1・長文読解
- `q2-cloze.pdf` — 大問2・空所補充
- `q3-grammar.pdf` — 大問3・文法

### その他

| ファイル名 | 意味 |
|-----------|------|
| `manual-p1.pdf` | 完全攻略マニュアル 第1部 |
| `manual-p2-vocab.pdf` | マニュアル 第2部（単語帳） |
| `mock-1.pdf` | 総合模試 第1回 |
| `vocab.pdf` | 単語・熟語帳 |
| `exam-2026-early.pdf` | 2026年一般選抜（前期日程） |
| `exam-2026-dept.pdf` | 2026年一般選抜（系統別日程） |
| `guide-2026.pdf` | 完全攻略（年度付き） |

### 追加例

```
bunkyo-u/q7-new-type.pdf
fukuoka-u/exam-2027-early.pdf
```

## サブフォルダ

大学によってサブフォルダを使ってよい（例: `fukuoka-u/past-question-pack/`）。
サブフォルダ名も英語・小文字・ハイフン区切りにする。

## 注意

- ファイル名を変えたら、HTML等のリンクも合わせて更新すること
- 大きなPDFを大量に置くとリポジトリが肥大化する。1ファイル数MBまでが目安
- 内容だけ差し替える場合は `-v2` などバージョンを付けるとCDNキャッシュ回避に安全
