# LEAP 空所補充テスト データ

LEAP改訂版 全2299語に対する 4択・空所補充問題の完成データ。スプレッドシートに貼ってそのまま使える。

## ファイル
- `LEAP_cloze_all.csv` … **本体**。全2299語ぶんの問題。ヘッダー付き。
- `cloze_sample_001-020.csv` … 手作りサンプル（No.1〜20、品質基準の見本）。
- `leap_words.tsv` … 見出し語マスター（No + 単語）。再生成・別テスト作成の元データ。

## 列の意味（LEAP_cloze_all.csv）
`No.,単語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`
- `日本語` … その問題で問うている語義の簡潔な説明（オリジナル表記）。
- `問題` … 空所 `(     )` を含む英文。
- `選択肢1〜4` … 4択。正解1つ＋ダミー3つ。
- `答え` … 正解の選択肢番号（1〜4）。位置は均等にシャッフル済み。
- `日本語訳` … 正解を入れた完成文の和訳。

## 品質方針
1. 各語の核心の語義（CEFR B1〜B2 レベルの最重要義）で出題。
2. **唯一解保証**：ダミー3つは空所に入れると文法破綻 or 明確に不自然になる語を選定（正解の同義語・別義で成立する語は不使用）。
3. 自然な英文・正確な和訳。例文・和訳・日本語の意味はすべて新規書き起こし（教材の定義文の転載なし）。

3段の検証パイプライン（作問 → 唯一解チェック → 自然さ/正確さチェック）を全語に適用。

## スプレッドシートでの使い方
1. Googleスプレッドシート等で「ファイル > インポート > アップロード」から `LEAP_cloze_all.csv` を読み込む（区切り＝カンマ）。
2. 範囲を絞って印刷シート/PDF化、または GAS で並べ替え・ランダム抽出して別形式テストを量産できる。

## 注意 / 今後
- 多義語でテスト語義がLEAPの想定とずれている語があれば、No.を指定すれば該当語だけ作り直せる。
- 同じマスターから「英→日4択」「日→英4択」「スペリング」等の別テストも追加生成可能。
