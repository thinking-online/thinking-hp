THE ONE REASON

英語が伸びない理由は、
たった1つ。

英語を、「左から右に」読めていない。

多くの受験生は、英文を「後ろから返り読み」している。だから読むのが遅く、長文で時間切れになり、選択肢をフィーリングで選ぶことになる。

単語も、文法も、そのために覚えているはずなのに── 肝心の「読む」という一点が、誰からも教われていない。ここが空白のまま量を積んでも、点数は止まったままです。

逆に言えば、「読み方」さえ入れ替われば、今持っている知識が一気に得点に変わる。CORE ENGLISH は、その一点だけを、最初から作り直す講座です。

THE CORE METHOD

「読める」を作る、
3つの核心。

CORE ENGLISH は、英語を「暗記科目」から「読解科目」へ作り変える。
その土台になるのが、この3つの核心です。

CORE 01

英単語の核心

「覚える単語」ではなく「読める単語」へ。語源とイメージから意味の核を押さえ、長文の中で一瞬で反応できる状態を作る。1ヶ月で1,000語を、忘れない形で入れる。

CORE 02

英文法の核心

文法問題を解くためではなく、英文を読むための文法。丸暗記ではなくイメージで理解し、2週間かけずに「読解で使える文法」を完成させる。

CORE 03

英文解釈の核心

構文を「左から右に」処理する技術。返り読みを捨て、スラスラ読んで意味が像として立ち上がる感覚を身につける。長文の得点が安定し始める分岐点。

本気で英語の基礎から固めて、「読み方」からフォーカスして、左から右にスラスラ読めて、点数が安定する方法を掴み取れ。
