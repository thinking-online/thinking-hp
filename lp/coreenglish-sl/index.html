ACT 03 解決策を見せる

WHAT IS CORE ENGLISH

Core Englishは、知識を配る講座ではありません。
知識を、脳内にインストールする方法を教える講座です。

私たちが徹底しているのは、たった1つの順番です。

① やり方

分野のコアを理解する

→
② 演習

理解を実戦に落とす

世の中の教材は、ほぼすべてが「演習」から始まります。
だから、やってもやっても身につきません。

野球で考えてみてください。
バッティングフォームが間違ったまま、毎日素振りを1000回しても、上達しません。
それどころか、間違ったフォームが癖になります。

英語も、まったく同じです。

だから、Core Englishは全5領域で「やり方」から入ります。
単語も、文法も、解釈も、長文も、リスニングも、例外なくです。

その5領域が、それぞれどう変わるのか。1つずつ見てください。

5 DOMAINS

英語は、5つの部品でできている。
Core Englishは、その全部を、正しい順番で組み立てます。

英語が伸びない生徒には、共通点があります。
5つの部品のうち、どれかが決定的に欠けているのに、本人がそれに気づいていない。

だから、単語が原因だと思って単語帳を替える。
文法が原因だと思って問題集を増やす。

でも、伸びない。原因の部品が、違うからです。

DOMAIN 01

英単語覚えるのをやめると、覚えられる

これまでのやり方
単語帳を1ページ目から順に、赤シートで隠して何周もする。

何が起きているか
単語帳に載っている日本語は、その単語の代表的な訳の1つにすぎません。
代表訳だけを頭に入れた状態で長文に入ると、知っているはずの単語が並んでいるのに、意味が通らない。
これが「単語は覚えたのに読めない」の正体です。この覚え方で通用するのは、学校の小テストまでです。

Core Englishのやり方

  • コア解説1つの単語に複数の意味が載っていても、核となる意味を理解すれば、個別に覚える必要はなくなります。leave なら「その場に留まらない」。この核が入っていれば、去る、置き忘れる、ある状態のままにしておく、が全部つながります。
  • イメージ解説日本語の訳語を見た瞬間に、絵が浮かぶ状態にします。英単語を日本語に変換して終わり、ではなく、日本語をさらにイメージに変換する。ここまでやるから、忘れません。
  • 実戦テスト 全50回覚えた気で終わらせません。長文の中で使える状態かどうかを、数字で確認します。
これまで
  • 情報量が多すぎる
  • 覚えてもすぐ忘れる
  • 模試で思い出せない
Core English
  • 覚えるのはコアだけ
  • 一度理解したら忘れない
  • 長文を読むときに、意味がスラスラ出てくる

DOMAIN 02

英文法暗記をやめると、覚える量が減る

これまでのやり方
関係代名詞をやって、仮定法をやって、分詞構文をやる。単元が終わったらチェックを入れて、次へ進む。

何が起きているか
文法を、バラバラの知識として覚えることになります。文法問題集は解けるようになります。ですが、長文の中では一切使えません。
実際の英文には「ここから仮定法です」という表示がないからです。

Core Englishのやり方
過去形のコアは、距離感です。これを理解すると、時制の過去形も、仮定法過去も、仮定法過去完了も、丸暗記せずに全部つながります。

  • 英文法コア講座 全100レッスン3000ページを超える、全イラスト解説
  • 実戦テスト 全12回専門用語で煙に巻きません。なぜそうなるのかを、小学生でも分かる図で説明します。
  • かかる期間は、12日間それで、高校英文法が終わります。覚える量は、むしろ減ります。
これまで
  • 「なぜ」の解説がないから丸暗記
  • 難しい専門用語で分かりにくい
  • 文法問題でしか使えない
Core English
  • コアから理解するので、脱・丸暗記
  • 小学生でも分かるイメージ図解
  • 長文でも使える文法力

DOMAIN 03

英文解釈「〇〇型」の暗記から抜ける

これまでのやり方
「〇〇の時はこう訳せ」「これは英文〇〇型」と、パターンを1つずつ覚えていく。

何が起きているか
解説を見た英文は読めます。ですが、初見の英文は読めません。知識が点のまま散らばっていて、線としてつながっていないからです。

Core Englishのやり方
英検、模試、共通テスト、二次試験。そのすべてに共通する「英文の絶対法則」から入ります。英語と日本語の決定的な違い。英語の絶対ルールである「補う」。これを映像で、視覚的に理解するところからスタートします。

  • 英文解釈 入門講座 全5レッスン品詞と文型 / 句 / 節 / 関係詞 / 特殊構文。5日間で、どの英文にも通じるコアが入ります。
  • 英文解釈 実践講座 全54レッスン100ページを超える全イラスト解説つき。25日間で、英文解釈が完成します。
これまで
  • 「〇〇型」として1つずつ丸暗記
  • 文字ばかりで、圧倒的に分かりにくい
  • 長い文章、複雑な文章に対応できない
Core English
  • 全範囲につながるコアだけを理解
  • 小学生でも分かるイメージ図解
  • 長文を、左から右にスラスラ読める

DOMAIN 04

長文読解現代文まで伸びる、論理の読み方

これまでのやり方
長文を解く。丸つけをする。解説の和訳と語彙と構造を読む。次の長文へ。

何が起きているか
市販の問題集の解説は、語彙解説と構造解説で終わっています。ですが、英文を読む目的は、筆者が伝えたいことを理解することです。
文章と文章のつながり。段落と段落の関係。どこが言い換えで、どこが具体例で、どこが譲歩なのか。この「読解」まで解説してある問題集は、ほとんど存在しません。

  • 論理読解講義 全5テーマ抽象から具体へ。言い換え。譲歩。5日間、1日1テーマ。
  • 長文演習2020の良質な英文について、1文1文がどうつながっているか、筆者がどう主張を展開しているかを、全文章分すべて解説します。
  • 実戦テスト 全20回構文を丸暗記しただけでは解けない、論理的に読めていないと答えが出ないオリジナル問題。

このやり方を身につけた生徒から、現代文の成績まで上がったという報告が毎年届きます。論理的に文章を読む力は、言語を問わないからです。

これまで
  • 読み方も分からず、ただひたすら読む
  • 語彙解説と構造解説だけ
  • ひたすら音読で、速読を目指す
Core English
  • 読み方と解き方を理解してから演習
  • 語彙／構造／読解の3点セット
  • 速読力がつく音読のやり方から入る

DOMAIN 05

リスニング聞き流しでは、一生伸びない

これまでのやり方
とりあえず英語を毎日聞く。共通テスト対策問題集を、直前に何度も聞く。

何が起きているか
アラビア語の発音ルールを何も知らない人が、アラビア語を1日100回聞き続けたら、いつか聞き取れるようになるでしょうか。
なりません。発音の仕組みを知らない音は、何回聞いても脳が処理できないからです。英語も、まったく同じです。

Core Englishのやり方
英語の発音の5大ルールから入ります。なぜその音がそう聞こえるのかを先に理解し、そのうえで実践に入ります。

  • リスニング文章講義 全30発音とリスニングの絶対ルールを30個、すべて解説します。
  • 完全オリジナル音声 全30実際の英文で、学んだルールがどう使われているかを確認します。
  • ディクテーションテスト 全30偏差値70超えを大量に輩出してきた、本物のトレーニング法です。「聞き取れているつもり」を、完全に潰します。
これまで
  • ただ闇雲に、ひたすら聞いて慣れろ
  • やった問題しか、聞き取れるようにならない
Core English
  • 発音のルールを理解して、実践する
  • 初めて聞く英文でも、聞き取れる

やり方を教える。そのうえで、演習をさせる。

人は、誰にも見られていない状態では続けられないようにできています。

03
