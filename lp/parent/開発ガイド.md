# THINKING保護者通信 — 開発・運用ガイド

保護者向け月刊通信サイトの追加・更新手順です。王様通信（`lp/king-letter-*`）と同じ仕組みで動きます。

## 公開URL

| ページ | URL |
|--------|-----|
| アーカイブ（バックナンバー一覧） | `https://thinking-online.com/parent` |
| 各号 | `https://thinking-online.com/parent-YYYYMM-NN` |
| 例：6月 vol.01 | `https://thinking-online.com/parent-202506-01` |

## フォルダ構成

```
lp/
├── parent/                    # アーカイブ（バックナンバー一覧）
│   ├── index.html
│   ├── styles.css
│   ├── assets/
│   └── 開発ガイド.md          # このファイル
└── parent-YYYYMM-NN/          # 各号（月1回追加）
    ├── index.html
    ├── styles.css
    ├── scripts.js
    └── assets/                # カバー画像・バナーなど
pdfs/
└── parent-YYYYMM-*.pdf        # まとめ・長文解説・コラムのPDF
```

## 公開・非公開の切り替え

**準備ができた号だけ**公開します。未公開の号はフォルダに置いておいても、次の2点をしない限りユーザーからは見えません。

1. `netlify.toml` にリダイレクトを追加しない
2. `lp/parent/index.html` のアーカイブにカードを載せない

### 新しい号を公開するとき

1. 号の中身（本文・PDF）を仕上げる
2. `netlify.toml` にリダイレクトを追加（下記テンプレート）
3. `lp/parent/index.html` の `issue-list` **一番上**にカードを追加
4. push → デプロイ

```toml
# LP: THINKING保護者通信 7月 vol.02 — /parent-202507-01 で公開
[[redirects]]
  from = "/parent-202507-01"
  to = "/lp/parent-202507-01/index.html"
  status = 200
[[redirects]]
  from = "/parent-202507-01/*"
  to = "/lp/parent-202507-01/:splat"
  status = 200
```

**現在公開中：** 6月号のみ（`/parent`・`/parent-202506-01`）

## 毎月の追加手順（約30分）

### 1. 新しい号フォルダを作る

前号をコピーしてリネームします。

```bash
cp -r lp/parent-202506-01 lp/parent-202507-01
```

命名規則：`parent-YYYYMM-NN`
- `YYYYMM` … 配信年月（例：202507）
- `NN` … その月の号数（通常は `01`）

### 2. 月次カバー画像を配置する

作成したその月のカバー画像を、新号フォルダに **`assets/img_cover.png`** として保存します。

```bash
# 例：7月号
cp ~/Desktop/parent-202507-cover.png lp/parent-202507-01/assets/img_cover.png
```

### 3. 号の中身を更新する

`lp/parent-202507-01/index.html` を開き、以下を書き換えます。

| 箇所 | 変更内容 |
|------|----------|
| `<title>` / OGタグ | 月・号数 |
| イントロ文言 | 必要に応じて |
| ヒーロー画像 | `assets/img_cover.png`（その月のカバー画像） |
| 目次（TOC） | 今月のコンテンツに合わせて |
| `#letter` | 今月の保護者通信本文・ハイライト |
| `#reading` | 英語読解特訓カード（Week 1〜N） |
| `#column` | 保護者コラム |
| `#notice` | お知らせ（`notice-item` ブロックをコピーして追加） |
| PDFリンク | `/pdfs/parent-202507-letter.pdf` など |

**英語長文を追加するとき**は、`reading-card` ブロックをコピーして Week 番号・タイトル・PDFパスを更新します。

### 4. PDFを配置する

`pdfs/` に以下の命名で置きます（push するだけで公開されます）。

```
pdfs/parent-202507-letter.pdf        # その月の保護者通信
pdfs/parent-202507-reading-w1.pdf    # 英語読解特訓 Week 1
pdfs/parent-202507-reading-w2.pdf    # 英語読解特訓 Week 2
…
pdfs/parent-202507-column.pdf        # 保護者コラム全文
```

### 5. netlify.toml にリダイレクトを追加

```toml
# LP: THINKING保護者通信 7月 vol.01 — /parent-202507-01 で公開
[[redirects]]
  from = "/parent-202507-01"
  to = "/lp/parent-202507-01/index.html"
  status = 200
[[redirects]]
  from = "/parent-202507-01/*"
  to = "/lp/parent-202507-01/:splat"
  status = 200
```

### 6. アーカイブにカードを追加

`lp/parent/index.html` の `issue-list` 内、**一番上**に新しい `issue-card` を追加します。

```html
<a class="issue-card" href="/parent-202507-01">
  <div class="issue-card__thumb">
    <img src="/parent-202507-01/assets/img_cover.png" alt="" …>
  </div>
  <div class="issue-card__body">
    <div class="issue-card__meta">
      <span class="issue-card__date">2026.07</span>
      <span class="issue-card__vol">VOL.02</span>
    </div>
    <h3 class="issue-card__title">第二号｜7月号</h3>
    <p class="issue-card__lead">リード文をここに</p>
  </div>
  <span class="issue-card__arrow" aria-hidden="true">›</span>
</a>
```

### 7. デプロイ

```bash
git add lp/parent-202507-01 lp/parent/index.html netlify.toml pdfs/
git commit -m "add parent letter 202507 vol.01"
git push
```

Netlify が自動デプロイします。

## 各号のコンテンツ構成

毎月、以下の **4項目** を掲載します。

| セクション | ID | 内容 | 必須 |
|-----------|-----|------|------|
| その月の保護者通信 | `#letter` | 今月の学びの様子・塾からのメッセージ（PDF） | ◎ |
| 英語読解特訓 | `#reading` | その月に扱った長文の解説PDF（Week 1〜N） | ◎ |
| 保護者コラム | `#column` | 家庭での声かけ・見守り方 | ○ |
| お知らせ | `#notice` | 夏期講習・面談・イベントなどのご案内 | ○ |

## 画像について

### 月次カバー画像（毎月必須）

各号のトップ画像は、**その月専用のカバー**を `assets/img_cover.png` として配置します。

```
lp/parent-202506-01/assets/img_cover.png  ← 6月号
lp/parent-202507-01/assets/img_cover.png  ← 7月号（翌月以降も同様）
```

| 項目 | 仕様 |
|------|------|
| ファイル名 | 常に `img_cover.png`（号フォルダごとに1枚） |
| サイズ | 1024×1024 px 推奨 |
| デザイン | 月ごとにイラスト・季節感を変える（6月号＝紫陽花など） |
| HTML | パスは `/parent-YYYYMM-01/assets/img_cover.png` のまま差し替え不要 |

**毎月の手順：**
1. その月用のカバー画像を作成
2. 新号フォルダの `assets/img_cover.png` に上書き保存
3. `index.html` の `alt` テキストだけ月に合わせて更新（例：`6月号` → `7月号`）
4. アーカイブ（`lp/parent/index.html`）のサムネイルは、各号の `img_cover.png` を参照するため自動反映

アーカイブページ上部のヒーロー（`lp/parent/assets/img_archive_hero.png`）は全号共通です。月次カバーとは別画像です。

## 王様通信との違い

| | 王様通信 | 保護者通信 |
|--|---------|-----------|
| 対象 | 受験の王様 LINE登録者（受験生中心） | THINKING通塾生の保護者 |
| デザイン | 緑×金（王様ブランド） | ネイビー×セージ（THINKINGブランド） |
| 主なコンテンツ | 特典PDF・動画・体験案内 | 保護者通信・英語読解特訓・コラム・お知らせ |
| 公開 | noindex（LINE限定感） | noindex（保護者限定） |

## トラブルシューティング

- **ページが 404** → `netlify.toml` のリダイレクトを確認
- **画像が表示されない** → パスが `/parent-202507-01/assets/...` 形式か確認（`/lp/` は付けない）
- **PDFが開けない** → `pdfs/` にファイルがあるか、ファイル名が HTML のリンクと一致しているか確認
