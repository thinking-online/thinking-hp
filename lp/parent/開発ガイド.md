# THINKING保護者通信 — 開発・運用ガイド

保護者向け月刊通信サイトの追加・更新手順です。王様通信（`lp/king-letter-*`）と同じ仕組みで動きます。

## 公開URL

| ページ | URL |
|--------|-----|
| アーカイブ（バックナンバー一覧） | `https://thinking-online.com/parent` |
| 各号 | `https://thinking-online.com/parent-YYYYMM-NN` |
| 例：6月 vol.01 | `https://thinking-online.com/parent-202506-01` |

## フォルダ構成

```
lp/
├── parent/                    # アーカイブ（バックナンバー一覧）
│   ├── index.html
│   ├── styles.css
│   ├── assets/
│   └── 開発ガイド.md          # このファイル
└── parent-YYYYMM-NN/          # 各号（月1回追加）
    ├── index.html
    ├── styles.css
    ├── scripts.js
    └── assets/                # カバー画像・バナーなど
pdfs/
└── parent-YYYYMM-*.pdf        # まとめ・長文解説・コラムのPDF
```

## 毎月の追加手順（約30分）

### 1. 新しい号フォルダを作る

前号をコピーしてリネームします。

```bash
cp -r lp/parent-202506-01 lp/parent-202507-01
```

命名規則：`parent-YYYYMM-NN`
- `YYYYMM` … 配信年月（例：202507）
- `NN` … その月の号数（通常は `01`）

### 2. 号の中身を更新する

`lp/parent-202507-01/index.html` を開き、以下を書き換えます。

| 箇所 | 変更内容 |
|------|----------|
| `<title>` / OGタグ | 月・号数 |
| イントロ文言 | 必要に応じて |
| ヒーロー画像 | `assets/img_cover.svg`（または .jpg） |
| 目次（TOC） | 今月のコンテンツに合わせて |
| `#summary` | 今月のまとめ本文・ハイライト |
| `#reading` | 英語長文カード（Week 1〜N） |
| `#column` | 保護者コラム |
| `#tips` | 見守り方のポイント |
| PDFリンク | `/pdfs/parent-202507-*.pdf` |

**英語長文を追加するとき**は、`reading-card` ブロックをコピーして Week 番号・タイトル・PDFパスを更新します。

### 3. PDFを配置する

`pdfs/` に以下の命名で置きます（push するだけで公開されます）。

```
pdfs/parent-202507-summary.pdf       # 今月のまとめ
pdfs/parent-202507-reading-w1.pdf    # 長文解説 Week 1
pdfs/parent-202507-reading-w2.pdf  # 長文解説 Week 2
…
pdfs/parent-202507-column.pdf        # 保護者コラム全文
```

### 4. netlify.toml にリダイレクトを追加

```toml
# LP: THINKING保護者通信 7月 vol.01 — /parent-202507-01 で公開
[[redirects]]
  from = "/parent-202507-01"
  to = "/lp/parent-202507-01/index.html"
  status = 200
[[redirects]]
  from = "/parent-202507-01/*"
  to = "/lp/parent-202507-01/:splat"
  status = 200
```

### 5. アーカイブにカードを追加

`lp/parent/index.html` の `issue-list` 内、**一番上**に新しい `issue-card` を追加します。

```html
<a class="issue-card" href="/parent-202507-01">
  <div class="issue-card__thumb">
    <img src="/parent-202507-01/assets/img_cover.svg" alt="" …>
  </div>
  <div class="issue-card__body">
    <div class="issue-card__meta">
      <span class="issue-card__date">2026.07</span>
      <span class="issue-card__vol">VOL.02</span>
    </div>
    <h3 class="issue-card__title">第二号｜7月号</h3>
    <p class="issue-card__lead">リード文をここに</p>
  </div>
  <span class="issue-card__arrow" aria-hidden="true">›</span>
</a>
```

### 6. デプロイ

```bash
git add lp/parent-202507-01 lp/parent/index.html netlify.toml pdfs/
git commit -m "add parent letter 202507 vol.01"
git push
```

Netlify が自動デプロイします。

## 各号のコンテンツ構成（推奨）

| セクション | 内容 | 必須 |
|-----------|------|------|
| 今月のまとめ | 学習ハイライト・塾からのお知らせ | ◎ |
| 英語長文解説 | その月に扱った長文の解説PDF | ◎ |
| 保護者コラム | 家庭での声かけ・見守り方 | ○ |
| 家庭での見守り方 | 4つのポイント（短く） | ○ |
| ご相談 | サポートページへの導線 | ○ |

## 画像について

初号はプレースホルダーSVGを使用しています。本番用には以下を差し替えてください。

- `img_cover.svg` → カバー画像（1024×576 推奨）
- `img_summary.svg` → まとめセクション用バナー
- `img_reading.svg` → 長文セクション用バナー
- `img_column.svg` → コラムセクション用バナー

王様通信と同様、Canva や Figma で作った JPG/PNG でも問題ありません。

## 王様通信との違い

| | 王様通信 | 保護者通信 |
|--|---------|-----------|
| 対象 | 受験の王様 LINE登録者（受験生中心） | THINKING通塾生の保護者 |
| デザイン | 緑×金（王様ブランド） | ネイビー×セージ（THINKINGブランド） |
| 主なコンテンツ | 特典PDF・動画・体験案内 | 月次まとめ・英語長文解説・コラム |
| 公開 | noindex（LINE限定感） | noindex（保護者限定） |

## トラブルシューティング

- **ページが 404** → `netlify.toml` のリダイレクトを確認
- **画像が表示されない** → パスが `/parent-202507-01/assets/...` 形式か確認（`/lp/` は付けない）
- **PDFが開けない** → `pdfs/` にファイルがあるか、ファイル名が HTML のリンクと一致しているか確認
