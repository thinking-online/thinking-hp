# 英文法ポラリス1　完成問題集（全375問）

高2〜高3の夏に「この一冊を極める」ための、問題編・解説編セット。
原稿（Googleドキュメント）の全375問をチェック・清書し、問題だけを並べた**問題編**と、
1問ずつ「なぜその答えになるか」まで解説した**解説編**に作り分けたものです。

## ファイル

| ファイル | 内容 |
|---|---|
| `index.html` | 表紙・入口（問題編／解説編へのリンク） |
| `mondai.html` | **問題編**（問題だけがずらっと並ぶ／印刷対応） |
| `kaisetsu.html` | **解説編**（正解＋完成文＋訳＋POINT＋誤答チェック／巻末に編集チェック記録） |

| `data/editorial-issues.json` | 原稿チェックで見つかった気づき一覧（27件） |
| `data/英文法ポラリス1_問題集マスター.csv` | **スプレッドシート管理用マスターCSV**（1問1行・問題＋解説まで全部入り） |

## マスターCSVについて（スプレッドシート管理用）

`data/英文法ポラリス1_問題集マスター.csv` は、Google スプレッドシート／Excel にそのまま
取り込める形式（UTF-8 BOM付き・全項目ダブルクオート囲み）です。**1問が1行**で、問題文から
解説まですべてが1枚のシートで管理できます。列は次のとおり：

| 列 | 内容 |
|---|---|
| 番号 | 001〜375 |
| チャプター / ユニット | 分類 |
| 形式 | 空所補充 / 整序 / 誤文訂正 / 記述 |
| 問題文 | 設問文（整序の語群〔①…〕もここに入ります） |
| 選択肢1〜6 | 選択肢（4択問題は1〜4、無い列は空欄） |
| 正解 | 正解番号（整序は並び順、誤文訂正は丸数字） |
| 正解の内容 | 正解の選択肢テキスト／完成文 |
| 完成文 | 正解を入れた英文 |
| 訳 | 日本語訳 |
| POINT | 急所の一言 |
| 解説 | 解説本文 |
| 誤答チェック・補足 | 各選択肢を潰す解説／整序の組み立て手順／訂正 |
| 備考 | 原稿の気づき（誤植メモなど） |

Google スプレッドシートには「ファイル → インポート → アップロード」でそのまま読み込めます
（区切り文字は自動判定でOK）。

HTMLはブラウザで開くだけで読め、そのまま印刷（PDF化）もできます。

## 構成

- **Chapter1 動詞関連**（時制／助動詞／仮定法／受動態）
- **Chapter2 準動詞**（不定詞／動名詞／分詞）
- **Chapter3 構造系**（関係詞／比較／疑問／否定／倒置・強調構文・語順）
- **Chapter4 品詞系**（前置詞／接続詞／冠詞・名詞／代名詞／形容詞／副詞）
- **Chapter5 文型**（第1〜第5文型の語法）

全30ユニット・375問。出題形式は「空所補充（4択）」「整序（並べ替え）」「誤文訂正」「記述」の4種類。

## 品質チェックについて

全問について、与えられた正解が英語として正しいかを検証しました。
その結果、**正解そのものが変わる重大な修正は1件のみ（問232：整序の解答が1桁不足していたのを補正）**で、
残りは誤植・不自然な英文・体裁の不統一といった軽微なものでした。
解説はいずれも正しい形に直したうえで執筆しています。詳細は解説編の巻末「編集チェック記録」を参照してください。
