# 単元別フォーム自動生成ツール（GAS・全問マーク式版）

マスターCSVを取り込んだスプレッドシートから、**単元（ユニット）ごとに Google フォーム
（自動採点クイズ）** を一括生成する Google Apps Script です。
全375問がマーク式（選択式）なので、**追加設定なしで全問が自動採点**されます。

## セットアップ（初回だけ）

1. **CSVをスプレッドシートに取り込む**
   `data/英文法ポラリス1_問題集マスター.csv` を Google スプレッドシートにインポート
   （ファイル → インポート → アップロード。1行目が見出し行になるようにします）。

2. **スクリプトを貼り付ける**
   スプレッドシートで「拡張機能 → Apps Script」を開き、`Code.gs` の中身を全部貼り付けて保存。

3. **メニューを出す**
   スプレッドシートを再読み込みすると、上部に「📝 フォーム作成」メニューが追加されます。

## 使い方

「📝 フォーム作成」メニューから選ぶだけ：

| メニュー | 動作 |
|---|---|
| 選択中の行の「単元」を作成 | いま選んでいるセルの行の単元だけ作成（いちばん手軽） |
| 単元名を指定して作成 | 単元名を入力して1つ作成（部分一致OK） |
| チャプターを指定して一括作成 | 例：「Chapter 1」で動詞関連の7単元をまとめて作成 |
| 全単元を一括作成（30単元） | 375問すべてを30フォームに一括生成 |
| データの読み込みを確認 | 何問・何単元読めているかを表示（動作確認用） |

- 初回実行時にGoogleアカウントのＯＡｕｔｈ承認を求められます（フォーム作成・Drive保存のため）。
- 作成したフォームは、`CONFIG.FOLDER_ID` で指定した **Drive フォルダ** に保存されます
  （既定でご指定のフォルダIDを設定済み。変更したい場合はこの値を差し替え）。
  フォルダIDを空にすると `FOLDER_NAME` のフォルダを自動作成して使います。
- 作成結果は、スプレッドシートに自動追加される **「フォーム一覧」** シートに次の列で記録されます：

  | 作成日時 | タブ | 対象No. | タイトル | 英文数 | 設問数 | 回答用URL | 編集用URL | 保存フォルダ |
  |---|---|---|---|---|---|---|---|---|

  - **タブ**：問題データを読み込んだシート（タブ）名
  - **対象No.**：その単元の問題番号の範囲（例：001〜013）
  - **英文数／設問数**：その単元の問題数（1問＝1英文＝1設問なので基本は同じ値）
  - **保存フォルダ**：保存先フォルダへのリンク（クリックで開けます）

## 出題形式（すべて選択式・自動採点）

| 形式 | フォームでの見え方 |
|---|---|
| 空所補充 | 選択肢1〜4から選ぶ |
| 整序（英文選択） | 日本語に合う正しい英文を4択から選ぶ |
| 誤文訂正 | 下線部①〜④（＝選択肢1〜4）から誤りを選ぶ |
| 記述（選択） | 空所に入る語を4択から選ぶ |

正解の選択肢には **完成文・訳・POINT・解説・誤答チェック** がフィードバックとして
付きます（`CONFIG.INCLUDE_EXPLANATION` でオフにも可）。

## 設定（Code.gs 冒頭の CONFIG）

| 項目 | 既定 | 説明 |
|---|---|---|
| `SHEET_NAME` | 空 | 問題データのシート名。空なら「問題」または先頭シートを自動判定 |
| `FOLDER_NAME` | 英文法ポラリス1_単元フォーム | 作成先フォルダ名 |
| `POINTS` | 1 | 1問あたりの配点 |
| `INCLUDE_EXPLANATION` | true | フィードバックに解説を載せる |
| `SHUFFLE` | false | 問題順をシャッフル |
| `COLLECT_EMAIL` | false | 回答者メールを収集 |

## 注意

- **必ずスプレッドシートのメニューから実行**してください。Apps Scriptエディタの
  ▶Run で直接実行すると、対象スプレッドシートを取得できずエラーになります。
- 全単元一括（30フォーム）は数分かかることがあります。GASの実行時間上限（6分）に
  かかる場合は、チャプター単位で分けて実行してください。
- 列の見出し名（番号 / ユニット / 問題文 / 選択肢1〜6 / 正解 …）は
  マスターCSVの見出しと一致している必要があります（列の並び順は問いません）。
