# 単元別フォーム自動生成ツール（GAS）

マスターCSVを取り込んだスプレッドシートから、**単元（ユニット）ごとに Google フォーム
（自動採点クイズ）** を一括生成する Google Apps Script です。

## セットアップ（初回だけ）

1. **CSVをスプレッドシートに取り込む**
   `data/英文法ポラリス1_問題集マスター.csv` を Google スプレッドシートにインポート
   （ファイル → インポート → アップロード。1行目が見出し行になるようにします）。

2. **スクリプトを貼り付ける**
   スプレッドシートで「拡張機能 → Apps Script」を開き、`Code.gs` の中身を全部貼り付けて保存。
   （マニフェストを編集できる場合は `appsscript.json` も合わせておくと権限承認がスムーズです）

3. **メニューを出す**
   スプレッドシートを再読み込みすると、上部に「📝 フォーム作成」メニューが追加されます。

## 使い方

「📝 フォーム作成」メニューから選ぶだけ：

| メニュー | 動作 |
|---|---|
| 選択中の行の「単元」を作成 | いま選んでいるセルの行の単元だけ作成（いちばん手軽） |
| 単元名を指定して作成 | 単元名を入力して1つ作成（部分一致OK） |
| チャプターを指定して一括作成 | 例：「Chapter 1」で動詞関連の7単元をまとめて作成 |
| 全単元を一括作成（30単元） | 375問すべてを30フォームに一括生成 |
| データの読み込みを確認 | 何問・何単元読めているかを表示（動作確認用） |

- 初回実行時にGoogleアカウントのＯＡｕｔｈ承認を求められます（フォーム作成・Drive保存のため）。
- 作成したフォームは Drive の **「英文法ポラリス1_単元フォーム」** フォルダに入ります。
- フォームの **回答用URL・編集用URL** は、スプレッドシートに自動追加される
  **「フォーム一覧」** シートに書き出されます。

## 出題形式ごとの扱い

| 形式 | フォーム化 | 採点 |
|---|---|---|
| 空所補充 | ラジオボタン（選択肢1〜6） | ✅ 自動採点 |
| 誤文訂正 | ラジオボタン（下線部①〜④を選択肢化） | ✅ 自動採点 |
| 整序 | 記述（番号の順で回答） | 送信後に正解＋解説を表示（自動採点は付きません※） |
| 記述 | 記述（英語で回答） | 送信後に正解＋解説を表示（自動採点は付きません※） |

※ GAS（FormApp）では記述式問題に「正解キー」をプログラムから設定できないため、
整序・記述は自動採点にはなりません。正解と解説は送信後のフィードバックで表示されます。
（フォームのUI上で手動で答えを設定すれば自動採点にできます。）

正解の選択肢・整序/記述の解答には、**完成文・訳・POINT・解説・誤答チェック** が
フィードバックとして付きます（設定でオフにも可）。

## 設定（Code.gs 冒頭の CONFIG）

| 項目 | 既定 | 説明 |
|---|---|---|
| `SHEET_NAME` | 空 | 問題データのシート名。空なら「問題」または先頭シートを自動判定 |
| `FOLDER_NAME` | 英文法ポラリス1_単元フォーム | 作成先フォルダ名 |
| `POINTS` | 1 | 1問あたりの配点 |
| `INCLUDE_EXPLANATION` | true | フィードバックに解説を載せる |
| `SHUFFLE` | false | 問題順をシャッフル |
| `COLLECT_EMAIL` | false | 回答者メールを収集 |

## 注意

- 全単元一括（30フォーム）は数分かかることがあります。GASの実行時間上限（6分）に
  かかる場合は、チャプター単位で分けて実行してください。
- 列の見出し名（番号 / ユニット / 形式 / 問題文 / 選択肢1〜6 / 正解 …）は
  マスターCSVの見出しと一致している必要があります（列の並び順は問いません）。
