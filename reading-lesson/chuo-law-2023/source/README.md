# chuo-law-2023 — 原典・参考解説（照合用 / Web非公開）

中央大学 法学部 2023年 大問8「よい聞き手（the good listener）」。

このフォルダは教材の逐語照合と論理検証の「正」を置く場所（`netlify.toml` でWeb非公開＝404）。
今回の教材は確認済みのため先行公開しているが、再検証・改訂に備えて以下を後追いで格納する：

- `passage.txt` … 元の英文（逐語）
- `questions.txt` … 設問文・選択肢（並び順そのまま）
- `answer-key.txt` … 正解キー（問1=c, 問2=b, … の一覧）
- `reference.md` / `*.pdf` … 参考解説
