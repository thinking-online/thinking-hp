# システム英単語 空所補充テスト データ

システム英単語の見出し語（「新」マーク=改訂追加語は除外）に対する 4択・空所補充問題の完成データ。

## ファイル
- `sistan_cloze_all.csv` … 本体。全1893問。ヘッダー付き。No.はシス単の番号（「新」除外のため欠番あり）。
- `sistan_words.tsv` … 見出しマスター（No + 見出し、新除外）。

## 列
`No.,単語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`

## 品質方針
正確さ／唯一解（ダミー3つは空所に入れると必ず破綻）／自然さ の3段検証を全見出しに適用。
例文・和訳・意味はすべて新規書き起こし（定義文の転載なし）。答え位置は1〜4に均等化。

## 注意
- 「新」付き134語は仕様により除外。No.はシス単の番号を保持（欠番あり）。
- 意味のズレがあればNo.指定で個別に作り直し可能。
