# 速読英熟語 空所補充テスト データ

全1084項目（熟語＋構文）に対する4択・空所補充問題の完成データ。

## ファイル
- `sokudoku_cloze_all.csv` … 本体。全1084問。ヘッダー付き。No.は通し番号(1〜1084)。
- `sokudoku_words.tsv` … 見出しマスター（No+見出し+意味、内部参照用）。

## 列
`No.,熟語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`

## 出題形式（熟語・構文に最適化）
- 熟語・句動詞 … 自然な英文中で熟語を空所化＋紛らわしい別熟語4択
- 構文・文法パターン（too ～ to do / no matter how / lest 等）… パターンを使った文で決め手の語を空所化＋構文語4択
- 会話・定型表現 … 短文/対話で空所化

正確さ／唯一解／自然さ の3段検証を全項目に適用。例文・和訳・意味はすべて新規書き起こし。答え位置は1〜4に均等化。
