# 夏休みに絶対に覚えるべき英単語300

これまで作成した単語帳（LEAP／鉄壁／システム英単語／ターゲット1900／英検準1級／英検準2級／英検2級）を
横断し、「複数の単語帳に共通して載っている＝重要度が高い」という基準で頻度上位300語を抽出したもの。

## ファイル
- `summer300_words.csv` … 暗記用リスト（No, 単語, 意味, 出現冊数）。出現冊数が多いほどコア。
- `summer300_cloze_all.csv` … 300語の4択空所補充テスト（既存の3段検証済み問題を流用）。

## 選定基準
7単語帳中、何冊に載っているかで重要度を測り、頻度＋出現の基礎度でランク付けした上位300語。
7冊共通=12語、6冊共通=272語、5冊共通から16語。
