# ターゲット1900 空所補充テスト データ

ターゲット1900 全1900語に対する4択・空所補充問題の完成データ。

## ファイル
- `target1900_cloze_all.csv` … 本体。全1900問。ヘッダー付き。No.はターゲット1900の番号。
- `target1900_words.tsv` … 見出しマスター（No+単語）。

## 列
`No.,単語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`

## 品質方針
正確さ／唯一解（ダミー3つは空所に入れると必ず破綻）／自然さ の3段検証を全語に適用。
例文・和訳・意味はすべて新規書き起こし。答え位置は1〜4に均等化。
