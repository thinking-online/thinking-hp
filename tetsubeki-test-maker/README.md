# 鉄壁 空所補充テスト データ

鉄壁英単語 全2176見出しに対する 4択・空所補充問題の完成データ。スプレッドシートに貼ってそのまま使える。

## ファイル
- `鉄壁_cloze_all.csv` … 本体。全2176問。ヘッダー付き。
- `tetsubeki_words.tsv` … 見出しマスター（No + 見出し）。

## 列
`No.,単語,日本語,問題,選択肢1,選択肢2,選択肢3,選択肢4,答え,日本語訳`

## 3タイプ対応
- 単語 … 1語空所＋1語4択
- 熟語・句動詞 … 句ごと空所＋紛らわしい句4択
- 会話表現 … 短い対話で空所＋表現4択

## 品質方針
正確さ／唯一解（ダミー3つは空所に入れると必ず破綻）／自然さ の3段検証を全見出しに適用。
crash/clash/crush, stare/gaze/glance 等の紛らわしい語は意味差が出る文脈で出題。
例文・和訳・意味はすべて新規書き起こし（定義文の転載なし）。

## 注意
- No.1351重複・1352欠番を整理（susceptible=1352）。
- 意味のズレがあればNo.指定で個別に作り直し可能。
